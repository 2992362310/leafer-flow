# Leafer Flow 验收清单

这份清单用于每次阶段性开发后做手动验收，重点覆盖用户可见链路和容易回归的编辑器能力。

## 启动与基础画布

- [ ] `pnpm run dev` 后页面可正常打开，无控制台启动错误。
- [ ] 画布区域可见，工具栏、图层面板、状态栏、日志面板显示正常。
- [ ] 缩放、放大、缩小、视图复位可正常工作。
- [ ] 网格吸附开关生效，关闭后拖拽不再吸附网格。

## 绘制能力

- [ ] 可绘制矩形、圆形、菱形、文本、箭头、自由绘制。
- [ ] 可绘制开始/结束、流程、判断、输入/输出、文档、数据库、子流程、连接点、泳道。
- [ ] 自由绘制线条平滑，首尾接近时可闭合。
- [ ] 绘制中按 `ESC` 或右键可取消当前绘制。
- [ ] 切换连接线类型后，新建连接线表现符合直线、折线或贝塞尔路径预期。

## 选择与编辑

- [ ] 点击元素可单选，空白处可取消选择。
- [ ] 框选可选中多个元素。
- [ ] `Ctrl+A` 可全选画布元素。
- [ ] `Delete` 可删除选中元素。
- [ ] 方向键可微移选中元素，`Shift+方向键` 可快速微移。
- [ ] 右键菜单可执行复制、粘贴、编组、取消编组、删除。

## 排版与层级

- [ ] 多选后可左对齐、居中对齐、右对齐、顶对齐、中线对齐、底对齐。
- [ ] 多选后可横向分布、纵向分布。
- [ ] 置顶、置底、上移一层、下移一层可改变遮挡关系。
- [ ] 锁定后元素不可被误操作移动或编辑。
- [ ] 隐藏后元素不可见，再次显示后位置和样式保持。

## 编组与剪贴板

- [ ] 多选元素后 `Ctrl+G` 可编组。
- [ ] 选中编组后 `Ctrl+Shift+G` 可取消编组。
- [ ] `Ctrl+C` / `Ctrl+V` 可复制粘贴普通图形。
- [ ] 复制粘贴包含连接线和标签的结构后，形状、连接关系和文字可恢复。

## 文件与导出

- [ ] 保存后重新加载，画布元素、位置、样式和层级保持。
- [ ] 刷新页面后可从自动恢复数据恢复最近画布。
- [ ] 导出 PNG 可得到非空图片。
- [ ] 导出 SVG 可得到可打开的矢量文件。

## 可用性检查

- [ ] 状态栏、右键菜单、日志面板没有乱码。
- [ ] 主要按钮在常见桌面宽度下不重叠、不溢出。
- [ ] 执行常见操作时日志记录清晰，不出现异常图标或空白消息。
- [ ] `pnpm run build` 可通过，且无大 chunk 警告。
